# Implementation Plan: Windows Installer (Inno Setup)

**Branch**: `006-windows-installer` | **Date**: 2026-06-13 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `specs/006-windows-installer/spec.md`

## Summary

إنشاء ملف `PLUS-Laundry-Setup.exe` بـ **Inno Setup 6** يحتوي جميع ملفات البرنامج من مجلد `release/`، يعرض واجهة عربية بسيطة لاختيار مجلد التثبيت، ثم ينسخ الملفات ويسجّل Scheduled Task للتشغيل التلقائي ويفتح البرنامج فور الانتهاء.

## Technical Context

**Language/Version**: Inno Setup Pascal Script 6.x + PowerShell 5.1

**Primary Dependencies**: Inno Setup 6 (build-time فقط، لا يُثبَّت على جهاز العميل)

**Storage**: لا تغيير في MySQL. مجلد `data/` يُنسخ مع `Flags: onlyifdoesntexist`

**Testing**: اختبار يدوي وفق `quickstart.md`

**Target Platform**: Windows 10/11 (64-bit)

**Project Type**: Packaging / Deployment tool

**Performance Goals**: التثبيت يكتمل في أقل من 60 ثانية. حجم Setup < 120 MB.

**Constraints**: لا متطلبات على جهاز العميل. لا اتصال إنترنت أثناء التثبيت.

**Scale/Scope**: تثبيت واحد لكل مغسلة.

## Implementation Phases

### Phase A: ملف Inno Setup (`installer/laundry.iss`)

**القسم [Setup]**:
```pascal
[Setup]
AppName=PLUS Laundry
AppVersion=1.0.12
AppPublisher=PLUS Systems
DefaultDirName={autopf}\PLUS\Laundry
DefaultGroupName=PLUS Laundry
OutputDir=..\dist
OutputBaseFilename=PLUS-Laundry-Setup
SetupIconFile=..\assets\icon\app.ico
Compression=lzma2/ultra64
SolidCompression=yes
PrivilegesRequired=admin
ArchitecturesInstallIn64BitMode=x64
RightToLeft=yes
```

**القسم [Languages]**:
```pascal
[Languages]
Name: "arabic"; MessagesFile: "compiler:Languages\Arabic.isl"
```

**القسم [Files]**:
```pascal
[Files]
; البرنامج الرئيسي — يُستبدل دائماً
Source: "..\release\laundry-app.exe"; DestDir: "{app}"; Flags: ignoreversion
Source: "..\release\mkcert.exe";      DestDir: "{app}"; Flags: ignoreversion

; السكريبتات
Source: "..\release\scripts\*"; DestDir: "{app}\scripts"; Flags: ignoreversion recursesubdirs

; بيانات أولية — لا تُستبدل إذا كانت موجودة
Source: "..\release\data\*"; DestDir: "{app}\data";
  Flags: ignoreversion recursesubdirs onlyifdoesntexist uninsneveruninstall

; مجلد ssl فارغ
Source: "..\release\ssl\*"; DestDir: "{app}\ssl";
  Flags: ignoreversion recursesubdirs onlyifdoesntexist uninsneveruninstall
```

**القسم [Run]** — تسجيل Task Scheduler وتشغيل البرنامج:
```pascal
[Run]
; إيقاف المهمة القديمة إذا وجدت
Filename: "powershell.exe";
  Parameters: "-NoProfile -ExecutionPolicy Bypass -Command ""schtasks /delete /tn LaundryPOS /f 2>$null""";
  Flags: runhidden waituntilterminated; StatusMsg: "جاري إعداد التشغيل التلقائي..."

; تسجيل Scheduled Task جديد
Filename: "powershell.exe";
  Parameters: "-NoProfile -ExecutionPolicy Bypass -File ""{app}\scripts\install-service.ps1"" -ExePath ""{app}\laundry-app.exe""";
  Flags: runhidden waituntilterminated; StatusMsg: "جاري تسجيل خدمة التشغيل التلقائي..."

; تشغيل البرنامج
Filename: "powershell.exe";
  Parameters: "-NoProfile -Command ""Start-ScheduledTask -TaskName LaundryPOS""";
  Flags: runhidden waituntilterminated nowait; StatusMsg: "جاري تشغيل البرنامج..."
```

**القسم [Icons]** — اختصار Startup:
```pascal
[Icons]
Name: "{userstartup}\PLUS Laundry"; Filename: "{app}\laundry-app.exe";
  WorkingDir: "{app}"; IconFilename: "{app}\laundry-app.exe";
  Comment: "PLUS Laundry POS System"
```

### Phase B: تعديل `package.json`

إضافة script:
```json
"build:installer": "\"C:\\Program Files (x86)\\Inno Setup 6\\iscc.exe\" installer\\laundry.iss"
```

### Phase C: إنشاء مجلد `installer/`

الملف الوحيد: `installer/laundry.iss`

### Phase D: التحقق من متطلبات الـ release

قبل `build:installer` يجب أن يكون `release\laundry-app.exe` موجوداً (ناتج `npm run build`).
إذا لم يوجد `.env` في `release/`، يُنسخ من `.env.example` في [Code] section.

## Risk & Mitigations

| الخطر | الحل |
|-------|------|
| Inno Setup غير مثبَّت على جهاز التطوير | يُثبَّت مرة واحدة من jrsoftware.org |
| Arabic.isl غير موجود | يأتي مع Inno Setup v6 افتراضياً |
| PowerShell ExecutionPolicy يمنع السكريبت | نستخدم `-ExecutionPolicy Bypass` في [Run] |
| المستخدم يشغّل المثبّت بدون Administrator | `PrivilegesRequired=admin` يجبر UAC تلقائياً |
| data/ يُحذف عند إعادة التثبيت | `Flags: onlyifdoesntexist uninsneveruninstall` يحميه |
