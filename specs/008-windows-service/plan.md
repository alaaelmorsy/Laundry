# Implementation Plan: Windows Service Deployment

**Branch**: `008-windows-service` | **Date**: 2026-06-13 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `specs/008-windows-service/spec.md`

## Summary

تحويل البرنامج من exe يُشغَّل يدوياً إلى Windows Service يبدأ تلقائياً مع الجهاز، مع Desktop Shortcut يفتح المتصفح مباشرة، وخدمة تحديث تلقائية تُبلَّغ المستخدم وتثبّت الإصدار الجديد بموافقته. MySQL يظل يدوياً بالكامل — لا تعديل فيه.

الأداة: **NSSM v2.24** لتسجيل الـ service. التحديث: **cron داخل Express** يفحص GitHub Releases كل 6 ساعات ويُبلَّغ عبر invoke API.

## Technical Context

**Language/Version**: Node.js (CommonJS) — نفس الكود القائم بدون تغيير + Inno Setup 6 (Pascal Script)

**Primary Dependencies**: NSSM v2.24 (exe مُضمَّن في installer)، `node-cron` (موجود بالفعل)، `https` (مدمج في Node) للتحقق من التحديثات وتحميل الـ installer

**Storage**: لا تغيير في MySQL. حالة التحديث in-memory فقط. لا جداول جديدة.

**Testing**: اختبار يدوي وفق `quickstart.md` — 6 سيناريوهات

**Target Platform**: Windows 10/11 (64-bit) — نفس الهدف القائم

**Project Type**: Packaging / Deployment + Feature Addition (update checker)

**Performance Goals**: الـ Service يبدأ خلال 60 ثانية من إقلاع Windows. فحص التحديث لا يؤثر على أداء البرنامج (background cron).

**Constraints**: لا اتصال إنترنت مطلوب لتشغيل البرنامج. MySQL يدوي — لا installer يلمسه. لا تعديل في كود Express أو المنطق القائم.

**Scale/Scope**: تثبيت واحد لكل مغسلة.

### Phase A: إعداد NSSM في الـ Installer

**الملفات**: `installer/laundry.iss` + `installer/nssm.exe`

**التغييرات في laundry.iss**:

1. إضافة `nssm.exe` في `[Files]`:
```pascal

```

2. تعديل `[Code]` — إيقاف قبل التثبيت، تسجيل وتشغيل بعده:
```pascal
procedure RegisterService(AppDir: String);
var ResultCode: Integer;
    NssmPath: String;
begin
  NssmPath := AppDir + '\nssm.exe';
  Exec(NssmPath, 'remove LaundryPlusApp confirm', '', SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Sleep(1000);
  Exec(NssmPath, 'install LaundryPlusApp "' + AppDir + '\laundry-app.exe"', '',
       SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Exec(NssmPath, 'set LaundryPlusApp AppDirectory "' + AppDir + '"', '',
       SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Exec(NssmPath, 'set LaundryPlusApp DisplayName مغسلة بلس', '',
       SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Exec(NssmPath, 'set LaundryPlusApp Start SERVICE_AUTO_START', '',
       SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Exec(NssmPath, 'set LaundryPlusApp AppExit Default Restart', '',
       SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Exec(NssmPath, 'set LaundryPlusApp AppRestartDelay 5000', '',
       SW_HIDE, ewWaitUntilTerminated, ResultCode);
  Exec(NssmPath, 'start LaundryPlusApp', '', SW_HIDE, ewWaitUntilTerminated, ResultCode);
end;

procedure CurStepChanged(CurStep: TSetupStep);
var ResultCode: Integer;
begin
  if CurStep = ssInstall then begin
    Exec(ExpandConstant('{app}\nssm.exe'), 'stop LaundryPlusApp', '',
         SW_HIDE, ewWaitUntilTerminated, ResultCode);
    Sleep(2000);
    Exec(ExpandConstant('{sys}\taskkill.exe'), '/F /IM laundry-app.exe', '',
         SW_HIDE, ewWaitUntilTerminated, ResultCode);
    Sleep(1000);
  end;
  if CurStep = ssPostInstall then begin
    RegisterService(ExpandConstant('{app}'));
  end;
end;

procedure CurUninstallStepChanged(CurUninstallStep: TUninstallStep);
var ResultCode: Integer;
begin
  if CurUninstallStep = usUninstall then begin
    Exec(ExpandConstant('{app}\nssm.exe'), 'stop LaundryPlusApp', '',
         SW_HIDE, ewWaitUntilTerminated, ResultCode);
    Exec(ExpandConstant('{app}\nssm.exe'), 'remove LaundryPlusApp confirm', '',
         SW_HIDE, ewWaitUntilTerminated, ResultCode);
  end;
end;
```

### Phase C: API Methods (4-Step Checklist)

#### الـ 3 Methods الجديدة

**Step 1 — `database/db.js`**: لا إضافة (in-memory).

**Step 2 — `server/invokeHandlers.js`**:
```javascript
const updateChecker = require('./update-checker');

// داخل switch(method):
case 'checkForUpdate':
  return { success: true, ...updateChecker.getState() };

case 'dismissUpdate':
  updateChecker.dismiss();
  return { success: true };

case 'installUpdate': {
  const st = updateChecker.getState();
  if (!st.available) return { success: false, message: 'لا يوجد تحديث متاح' };
  await updateChecker.downloadInstaller();
  const st2 = updateChecker.getState();
  if (!st2.localPath) return { success: false, message: 'فشل تحميل التحديث' };
  const { spawn } = require('child_process');
  spawn('cmd', ['/c', 'start', '', st2.localPath], { detached: true, stdio: 'ignore' }).unref();
  return { success: true };
}
```

### Phase D: Update Banner UI

**`assets/update-banner.js`**:
```javascript
(function () {
  'use strict';
  const POLL_INTERVAL = 5 * 60 * 1000;

  function injectBanner() {
    const div = document.createElement('div');
    div.id = 'updateBanner';
    div.className = 'update-banner hidden';
    div.innerHTML = `
      <div class="update-banner-content">
        <span class="update-icon">🔔</span>
        <span id="updateBannerText"></span>
        <button id="btnInstallUpdate" onclick="handleInstallUpdate()">تثبيت الآن</button>
        <button id="btnDismissUpdate" onclick="handleDismissUpdate()">تذكير لاحقاً</button>
      </div>`;
    document.body.prepend(div);
  }

  async function check() {
    if (!window.api || !window.api.checkForUpdate) return;
    try {
      const res = await window.api.checkForUpdate();
      if (res.available) showBanner(res.version, res.releaseNotes);
    } catch (_) {}
  }

  function showBanner(version, notes) {
    const banner = document.getElementById('updateBanner');
    if (!banner) return;
    document.getElementById('updateBannerText').textContent =
      `يتوفر إصدار جديد: ${version}`;
    banner.classList.remove('hidden');
  }

  window.handleInstallUpdate = async function () {
    document.getElementById('btnInstallUpdate').disabled = true;
    document.getElementById('updateBannerText').textContent = 'جاري تحميل التحديث...';
    await window.api.installUpdate();
  };

  window.handleDismissUpdate = async function () {
    await window.api.dismissUpdate();
    document.getElementById('updateBanner').classList.add('hidden');
  };

  document.addEventListener('DOMContentLoaded', () => {
    injectBanner();
    check();
    setInterval(check, POLL_INTERVAL);
  });
})();
```

## Risk & Mitigations

| الخطر | الحل |
|-------|------|
| NSSM يحتاج Admin | `PrivilegesRequired=admin` في laundry.iss |
| MySQL لم يبدأ عند تشغيل الـ Service | Express connection pool يعيد المحاولة تلقائياً |
| GitHub API rate limit (60/h بدون token) | 4 requests/يوم فقط — بعيد عن الـ limit |
| انقطاع الإنترنت أثناء التحميل | catch صامت — لا رسالة خطأ للمستخدم |
| الـ installer القديم لا يحتوي nssm.exe | لا يؤثر — المستخدم يستخدم الـ installer الجديد |
| اسم الـ Service تغيّر من تثبيت قديم | RegisterService يبدأ بـ `nssm remove` احتياطي |
| تعارض منفذ 3443 | لا تغيير — نفس سلوك الإصدار القائم |
