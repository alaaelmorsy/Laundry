# Quickstart: Validate Print Font Weight Fix

## Prerequisites

- Chrome browser مثبّت
- البرنامج يعمل محلياً

## Test Scenarios

### 1. التحقق من فاتورة (invoices)
1. افتح شاشة الفواتير
2. اضغط على أي فاتورة لفتحها
3. اضغط `Ctrl+P` لفتح معاينة الطباعة
4. **المتوقع**: النص يظهر أسود وعريض في المعاينة

### 2. التحقق من التقرير اليومي (daily-report)
1. افتح شاشة التقرير اليومي
2. اضغط `Ctrl+P`
3. **المتوقع**: جميع الأرقام والنصوص تظهر أسود وعريض

### 3. التحقق من تقرير العمال (worker-report)
1. افتح شاشة تقرير العمال
2. اضغط `Ctrl+P`
3. **المتوقع**: أسماء العمال والأرقام تظهر واضحة وأسود

### 4. اختبار شامل
| الشاشة | النتيجة المتوقعة |
|--------|-----------------|
| invoices | نص أسود عريض |
| daily-report | نص أسود عريض |
| period-report | نص أسود عريض |
| worker-report | نص أسود عريض |
| subscriptions-report | نص أسود عريض |
| all-invoices-report | نص أسود عريض |
| pos (فاتورة) | نص أسود عريض |

## Verify No Regressions

- الشاشات تعمل بشكل طبيعي على الشاشة (خارج وضع الطباعة)
- معاينة الطباعة تُظهر تخطيطاً سليماً (لا تداخل في العناصر)
