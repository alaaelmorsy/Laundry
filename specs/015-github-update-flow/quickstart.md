# Quickstart: التحقق من نظام التحديث الجديد

**Feature**: 015-github-update-flow

## المتطلبات

- البرنامج يعمل (`npm start` أو الـ exe)
- الوصول إلى شاشة الإعدادات → تبويب "التحديثات"
- قاعدة بيانات MySQL متصلة
- اتصال إنترنت

---

## السيناريو 1: الدعم الفني منتهٍ

**الإعداد:**
```sql
UPDATE app_settings SET support_expiry_date = '2020-01-01' WHERE id = 1;
```

**الخطوات:**
1. افتح الإعدادات → تبويب التحديثات
2. اضغط "فحص التحديثات"

**النتيجة المتوقعة:**
- رسالة خطأ: "انتهت فترة الدعم الفني — يرجى تجديد الدعم للحصول على التحديثات"
- لا يحدث اتصال بـ GitHub
- لا يظهر زر التحميل

---

## السيناريو 2: الدعم ساريٍ، البرنامج محدَّث

**الإعداد:**
```sql
UPDATE app_settings SET support_expiry_date = '2027-12-31' WHERE id = 1;
```
(تأكد أن `package.json` يحتوي نفس الإصدار الموجود على GitHub أو أحدث)

**الخطوات:**
1. اضغط "فحص التحديثات"

**النتيجة المتوقعة:**
- رسالة: "البرنامج محدَّث ✓"
- لا يظهر زر التحميل

---

## السيناريو 3: يوجد تحديث جديد — تحميل وتثبيت (Golden Path)

**الإعداد:**
```sql
UPDATE app_settings SET support_expiry_date = '2027-12-31' WHERE id = 1;
```
(تأكد أن GitHub يحتوي إصداراً أحدث من `package.json`)

**الخطوات:**
1. اضغط "فحص التحديثات"
2. انتظر ظهور بطاقة التحديث مع معلومات الإصدار الجديد
3. اضغط "تحميل التحديث"
4. راقب شريط التقدم — يجب أن يعرض MB مثل `12.3 MB / 45.0 MB`
5. انتظر اكتمال التحميل (الشريط يصل 100%)
6. اضغط "تثبيت الآن" (يجب أن يتفعّل فقط بعد الاكتمال)

**النتيجة المتوقعة:**
- الخطوة 3: يظهر شريط التقدم فوراً ويبدأ من `0 MB / X MB`
- الخطوة 4: الأرقام تتحدث كل ثانية تقريباً
- الخطوة 5: زر "تثبيت الآن" يتفعّل، زر "تحميل التحديث" يتعطّل
- الخطوة 6: البرنامج يُغلق وتبدأ عملية التثبيت (نافذة المثبّت تظهر أو تعمل في الخلفية)

---

## السيناريو 4: انقطاع الإنترنت أثناء التحميل

**الإعداد:** افصل الإنترنت أثناء التحميل.

**النتيجة المتوقعة:**
- رسالة خطأ واضحة (مثل "انتهت مهلة التنزيل" أو "فشل التنزيل")
- شريط التقدم يتوقف
- يمكن الضغط على "فحص التحديثات" مجدداً

---

## السيناريو 5: الضغط على "تثبيت الآن" قبل اكتمال التحميل

**الخطوات:** اضغط "تثبيت الآن" فور بدء التحميل (أو بعده مباشرة).

**النتيجة المتوقعة:**
- الزر معطّل (لا يستجيب) حتى يكتمل التحميل

---

## التحقق من الـ API مباشرة

للتحقق من الـ API عبر المتصفح (DevTools Console):

```js
// فحص التحديثات
window.api.checkForUpdate({ force: true }).then(console.log)

// بدء التحميل
window.api.downloadUpdate().then(console.log)

// مراقبة التقدم
let t = setInterval(() => window.api.getUpdateProgress().then(r => {
  console.log(r.downloadedBytes, '/', r.totalBytes, '—', r.percent + '%', r.downloadDone ? 'DONE' : '');
  if (r.downloadDone) clearInterval(t);
}), 1000)

// تثبيت (بعد اكتمال التحميل فقط)
window.api.installUpdate().then(console.log)
```
