# Laundry App

Laundry Management System (Node.js + Express + TailwindCSS).

## Requirements

- Node.js (LTS recommended)
- npm
- MySQL (if configured in `.env`)

## Setup

```bash
npm install
```

- Copy `.env.example` to `.env` and fill the required values.

## Run

```bash
npm run start
```

App/server entry: `server/index.js`.

## Tailwind CSS

- Build once:

```bash
npm run build:css
```

- Watch:

```bash
npm run watch:css
```
